# 🏍 Motorbike Info Generator

##  Overview
* CLI tool built in Python
* Generates detailed insights for **real + fictional motorbikes**
* Uses an AI-style function (`call_gpt`)

###  Usage

```bash
python your_file_name.py
```
* Enter any motorbike name when prompted

#### Output Details
* Model history
* Engine specifications
* Top speed & mileage
* Design & features
* Price range
* Ideal user profile


 ##### AI Behavior

* Built-in fallback ensures smooth execution 
* Does not produce real AI-generated content by default 
* To enable full functionality, connect `call_gpt` to an API (e.g., OpenAI)
